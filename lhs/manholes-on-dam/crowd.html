# 人上一百

《人上一百》即官方的《梦陨春宵》剧本，属于中等难度剧本。

可查看《钟楼百科》的有关内容：剧本-梦陨春宵 (opens new window)

# 四有人员

# 假期刊（钟表匠）：

在你的首个夜晚，你会得知团长与团员之间最近的距离。（邻座的玩家距离为 1）

背景设定：民科是假期刊的常客，因此假期刊的工作人员知道两个民科的信息。

# 井盖专家（筑梦师）

每个夜晚，你要选择除你及猴山游客以外的一名玩家：你会得知一个 132 角色和一个 826 角色，该玩家是其中一个角色。

背景设定：井盖专家是编造的人物，设定为成电专家团队里专门前往井盖观察 826 的人员，但得到的信息往往不太准确。

# 刘彻东（舞蛇人）

每个夜晚，你要选择一名存活的玩家：如果你选中了团长，你和他交换角色和阵营，然后他精神错乱。

背景设定：刘彻东也是川大中文系，其身份说不准，因此可能加入 826。

# 绮梦璇（数学家）

每个夜晚，你会得知有多少名玩家的能力因为其他角色的能力而未正常生效。（从上个黎明到你被唤醒时）

背景设定：绮梦璇是民科吧见闻录特别篇的作者，对民科和反民科的意外事件非常熟悉。

# 雪时晴（卖花女孩）

每个夜晚（第一夜除外），你会得知在今天白天时是否有团长投过票。

背景设定：雪老师当过小吧主，揭露过雷绍武无故封禁吧友的罪证。

# 魔化希德（城镇公告员）

每个夜晚（第一夜除外），你会得知在今天白天时是否有团员发起过提名。

背景设定：魔化希德是雷吧老吧友，可以辨认出猴山 7 及其小号。

# 春小树（神谕者）

每个夜晚（第一夜除外），你会得知有多少名进盒的玩家是 826。

背景设定：春小树是民科吧吧主，知道民科的动态。

# 万斯教授（博学者）

每个白天，你可以私下询问说书人以得知两条信息：一个是正确的，一个是错误的。

背景设定：万斯教授是吧友编造的专家教授，他有知有德，但也受到了雷氏歪理的影响。

# 沃森教授（女裁缝）

每局游戏限一次，在夜晚时，你可以选择除你以外的两名玩家：你会得知他们是否为同一阵营。

背景设定：沃森教授是知名生物学家，他可以比对两个人的雷氏基因网络是否一致。

# 猴督（哲学家）

每局游戏限一次，在夜晚时，你可以选择一个 132 角色：你获得该角色的能力。如果这个角色在场，他看不清楚。

背景设定：猴督学习能力极强，且在猜匿名中能够很好地伪装成其他四有人员。

# 蓝色旋律（艺术家）

每局游戏限一次，在白天时，你可以私下询问说书人一个是非问题，你会得知该问题的答案。

背景设定：蓝色旋律是最爱春雷吧吧主，掌握重要的信息。

# 氢氦锂铍硼（杂耍艺人）

在你的首个白天，你可以公开猜测任意玩家的角色最多五次。在当晚，你会得知猜测正确的角色数量。

背景设定：公 审 赵 明 毅。

# 深海小贝壳（贤者）

如果团长将你批斗进盒，在当晚你会被唤醒并得知两名玩家，其中一名是批斗使你进盒的那个团长。

背景设定：深海小贝壳知道自己被雷绍武封禁了，曾经换小号试图举报雷绍武。

# 四无人员

# nideqiancou（畸形秀演员）

如果你对自己是四无人员疯狂表现，你可能被处决。

背景设定：nideqiancou 表现疯狂，被雷吧驱逐出去了。

# 园长（心上人）

当你进盒时，会有一名玩家开始看不清楚。

背景设定：发情是错误的，不明白个？

# 乐山大佛（理发师）

如果你进盒，在当晚团长可以选择两名玩家（不能选择其他团长）交换角色。

背景设定：826 砸坏乐山大佛，表现疯狂！

# 第一种人（呆瓜）

当你得知你进盒时，你要公开选择一名存活的玩家：如果他是 826，你的阵营落败。

背景设定：不明白的都是两种人！

# 团员

# 邦钦孙（镜像双子）

你与一名对立阵营的玩家互相知道对方是什么角色。如果其中的 132 被处决，826 阵营获胜。如果你们都存活，132 阵营无法获胜。

背景设定：邦钦孙经常把自己的贴吧昵称改成反民科人员的贴吧昵称，并导致了一些人分不清。

# 郭德胜（女巫）

每个夜晚，你要选择一名玩家：如果他明天白天发起提名，他进盒。如果只有三名存活的玩家，你失去此能力。

背景设定：郭德胜对部分反民科人员进行网暴，一些突出的反民科因此退网。

# 永丽照相馆（洗脑师）

每个夜晚，你要选择一名玩家和一个 132 角色。他明天白天和夜晚需要对自己是这个角色表现疯狂，不然他可能被处决。

背景设定：雷氏理论制作的照片，可以让人误以为自己是照片中的那个角色！（编造的故事小说）

# 假证贩子（麻脸巫婆）

每个夜晚（第一夜除外），你要选择一名玩家和一个角色，如果该角色不在场，他变成该角色。如果因此创造了一个团长，当晚的进盒由说书人决定。

背景设定：假证贩子在九眼桥贩卖假证，可以改变顾客的角色。

# 团长

# 雷天使（方古）

每个夜晚（第一夜除外），你要选择一名玩家：他进盒。

因该能力进盒的四无人员改为变成 826 阵营的雷天使且你代替他进盒，但每局游戏仅能成功转化一次。 [游戏初始配置改变：+1 四无人员]

背景设定：雷天使给四无人员传播真理！

# 雷老狗（亡骨魔）

每个夜晚（第一夜除外），你要选择一名玩家：他进盒。

因你进盒的团员保留他的能力，且与他邻近的两名四有人员之一精神错乱。[游戏初始配置改变：-1 四无人员]

背景设定：雷老狗对人狂吠，132不和无知无赖争论，自封了；团员表现出疯狗本性，因此能力保留且跟着雷绍武狂吠，使身边的四有人员不堪其扰。

# 王金甲（诺-达鲺）

每个夜晚（第一夜除外），你要选择一名玩家：他进盒。

与你邻近的两名四有人员精神错乱。

背景设定：王金甲整天复读，四有的邻居不堪其扰，精神错乱。

# 何松昂（涡流）

每个夜晚（第一夜除外），你要选择一名玩家：他进盒。

四有人员的能力都会产生错误信息。

如果白天没人被处决，826 阵营获胜。

背景设定：何松昂精神作法，给出错误信息。如果白天没人被处决了，何松昂精神胜利，宣布自己获胜了。

# 行动顺序

5、6 人时，没有团员信息和团长信息。

  • 第一夜： 团员获得信息-团长获得信息-猴督-刘彻东-邦钦孙-郭德胜-永丽照相馆-假期刊-井盖专家-沃森教授-绮梦璇
  • 其他夜晚： 猴督-刘彻东-郭德胜-永丽照相馆-假证贩子-雷天使-雷老狗-王金甲-何松昂-保卫处-园长-深海小贝壳-井盖专家-雪时晴-魔化希德-春小树-沃森教授-万斯教授-绮梦璇
