# 两个游戏群

  • 黄田坝： 931902336（不在坝桥一线时可进行猜匿名活动）
  • 九眼桥： 931949586（不在坝桥一线时可进行雷猴汤活动）

# 游戏配置及流程

# 游戏开始

游戏开始时随机把每个玩家分配到两个游戏群之一，并为玩家分发角色牌。任何玩家不可以与另一个游戏群内的玩家交谈。

然后，进入第一轮游戏。

# 提名队长

第一轮开始时，每个游戏群的玩家提名一个队长。仅第一个提名有效。

# 自由发言

任何一轮中，在人质确定前，玩家可在群内自由发言。

# 身份/阵营的共享和单向展示

自由发言阶段进行。

  • 身份/阵营共享： 私聊主持人，申请与某玩家共享身份/阵营。如果对方同意，则你们得知对方的身份/阵营。
  • 身份/阵营单向展示： 私聊主持人，向某玩家展示身份/阵营。也可选择向群内全体玩家展示。

# 队长的替换

一个房间的队长会保持队长身份，除非有以下两个事件的其中之一发生：

  1. 队长想把领导权给另一个玩家。 如果当前的队长不想要再当队长了(通常是队长自己想要成为人质)，那么该队长只需要把队长牌给另一个玩家即可。收到队长牌的玩家可以接受或是拒绝这张队长牌。如果该玩家拒绝，原本的队长还是必须继续当队长。如果该玩家接受，他就成为队长，收下队长牌，然后直到下一轮之前，他都不能把队长牌归还给前一位队长（即不能收了又反悔）。

  2. 夺取队长地位。 任何时候，只要有某个玩家不希望当前的队长继续担任队长，他可以向主持人申请替换队长（可以替换成自己或者他人），然后开始一轮投票。若超过半数玩家同意队长替换，则队长替换成功。

# 选择人质

自由发言阶段结束后，队长公开选择一名玩家成为人质（不能选择自己；一旦选定后不能更改）。待两群都选定人质后，本轮结束，人质前往另一个房间，下一轮开始。

每局游戏一般可进行 3 轮或 5 轮。

# 游戏结束

所有轮次均结束后，游戏结束，根据各玩家的技能描述判定胜负。

# 角色一览

  1. 科奴 132： 结束时不被电死，132 胜利。
  2. 雷绍武 826： 结束时接通零线，自己房间所有人都死，电死科奴 826 胜利。

  1. 黄新卫 第三方： 自己不死，刘武青被电死，则自己胜利。（设定参考 民科内斗 (opens new window)）
  2. 刘武青 第三方： 自己不死，黄新卫被电死，则自己胜利。（设定参考 民科内斗 (opens new window)）

  1. 第一种人 第三方： 自己与科奴结束时在一起，且第二种人不在，则自己胜利。
  2. 第二种人 第三方： 自己与科奴结束时在一起，且第一种人不在，则自己胜利。

  1. 周书记 132： 开局获得效果：结束时 132 失败，与科奴分享身份后解除。（设定是递给科奴推荐信，防止被批斗）
  2. 何文涛 826： 开局获得效果：结束时 826 失败，与雷绍武分享身份后解除。（设定是帮助雷绍武修理零线）

  1. 鱼箱 132： 如果和雷绍武分享身份牌，游戏立即结束。132 胜利。（设定参考雷猴杀）
  2. 石安迪 826： 如果你和科奴分享身份牌，游戏立即结束。826 胜利。（设定参考雷猴杀）

  1. 四无人员 132： 若科奴被埋葬，则你成为科奴；若科奴在场，则你视为普通的 132 成员，没有特殊能力。
  2. 猴山 7 826： 若雷绍武被埋葬，则你成为雷绍武；若雷绍武在场，则你视为普通的 826 成员，没有特殊能力。

  1. 132 新员工 132： 游戏开始时进入“青涩”状态。进入“青涩”状态。处于青涩状态的玩家不可向别人展示身份/阵营，必须拒绝有共享邀请。
  2. 826 新团员 826： 游戏开始时进入“青涩”状态。进入“青涩”状态。处于青涩状态的玩家不可向别人展示身份/阵营，必须拒绝所有共享邀请。

  1. 九头蛇 第三方： 结束时猜测被埋葬的牌是什么，如果正确，则你同时获胜。
  2. 投机电子 第三方： 在主持人宣布结果前，猜测以下三种情况之一：132 胜利、826 胜利、132 和 826 都失败。如果猜测正确，你同时获胜。
  3. 丧葬一条龙 第三方： 当其他玩家向你展示其角色/阵营时，必须交换双方的角色卡。游戏结束时，直接失败。
  4. 零度君 第三方： 游戏结束时，只要雷绍武与科奴都向你展示过身份（即使这名玩家之后失去了此身份），你同时获胜。
  5. 管理人员 第三方： 游戏结束时，如果你最终作为队长，则你同时获胜。
