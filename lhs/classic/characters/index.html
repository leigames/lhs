# 角色一览

本部分将列出雷猴杀游戏所有已使用的角色及其技能。

包括但不限于以下人员参与了角色的创作，排名不分先后：

  • 虎爪吃布丁
  • 莉莉艾
  • Sxlzr444
  • 连保卡璞鳍鳍
  • Unbroken
  • 魔化希德
  • Mono

# 技能术语说明

以下术语描述技能的性质，将在角色技能介绍中多次出现。

# 锁定技

当条件满足时自动发动的技能。不受玩家控制。

# 限定技

一局游戏中只能使用有限次数的技能，默认为 1 次。

# 触发技

一局游戏中只能生效有限次数的技能，默认为 1 次。一般情况下，你可以得知你的技能是否已经生效。

# 觉醒技

当条件满足时自动发动、改变角色的某种状态（如获得某新技能）且不可逆转的技能。

# 转换技

该技能由多个子技能构成。第一次响应该技能时，只能使用第一个子技能，第二次用第二个，……，如此循环往复。

# 阵营技

一局游戏只能发动一次，发动条件由投票决定的技能。

  • 132 阵营技发动条件： 在白天，不小于存活玩家的一半选择发动。
  • 826 阵营技发动条件： 在黑夜，不小于存活 826 的一半选择发动。
