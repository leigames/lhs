# 132 阵营角色

# 四无人员

  • 无特殊技能。

# 成电专家

  • 查验： 每天夜晚可以选择一名角色，查验其阵营是否为 826。

# 周书记

  • 推荐信： 限定技 当你持有推荐信时，每天夜晚可以知道即将被批斗出局的角色，然后可以使用推荐信防止其出局。但如果其同时被邱白林守护，则仍然出局。不能和扣饭同时使用。
  • 扣饭： 限定技 夜间可以对一名角色扣饭，令其出局。不能和【推荐信】同时使用。

# 邱白林

  • 守护： 每天夜晚可以选择一名角色进行守护，使其不会被批斗出局。但如果该玩家同时被周书记使用【推荐信】，则仍然出局。不能连续两晚守护同一名角色。

# 老卞

  • 大字报： 限定技 如果你被批斗或驱逐出局，可以写出大字报使另一名角色被驱逐。但是如果被扣饭出局，则不能使用该技能。

说明

若场上同时存在雷副教授，则其他玩家只能知道某玩家出局后发动了【大字报】技能，不知道该玩家是老卞还是雷副教授。

注意

老卞或雷副教授出局后不能发动【大字报】技能的情形很多，按照板子规则自行判断即可。其他类似技能同理。

# 鱼箱

  • 质疑： 限定技 白天可以亮出自己的身份，然后选择一名角色，对其身份进行质疑。若其为 826，则其出局，然后进入黑夜；若其为 132，则你出局，白天继续。

# 第一种人

  • 愚昧/自大： 觉醒技 当你白天被投票指定为驱逐对象时，你亮明身份而不被放逐，此后你可以在夜间发言，但白天失去投票资格。

说明

若场上同时存在天龙，则其他玩家只能知道某玩家“亮明身份免除放逐”，不知道该玩家是第一种人还是天龙。

# 第二种人

  • 盒中洞天： 锁定技 你无法出局，但在胜负判定时视为你已经出局。

说明

  1. 若你被指定为大字报的目标，则认定为大字报技能没有发动，主持人也不会公布写大字报的情况。
  2. 若你在白天出局，则主持人仅提示你无法出局，而不翻牌展示身份。

# 零度君

  • 见闻录： 每晚可以得知前一天（夜晚和白天）所有出局角色的具体身份。

# 杜自誊

  • 团队： 你与成电专家见面，晚上可以一起查验一人。但是若成电专家出局，你不可以单独查验。
  • 诈骗： 限定技 若你和成电专家查验出 826，你可以使该 826 立即出局，且其出局时不可以使用任何技能。

说明

有时候会规定杜自誊可以从当前查验出的所有 826 中选择一人诈骗出局。

某些板子中，查验到的 826 不一定是真正的 826。如果诈骗到假 826，则按板子规则处理。

# 弋之狐

  • 嘲讽： 每天夜晚可选择一名角色，对其进行嘲讽。第二天白天放逐投票时，被嘲讽的人票数+1。不能连续两晚选择同一个嘲讽对象。

# 吧主

  • 封禁： 触发技 夜间你可以选择一名玩家，使其当晚无法发动任何技能。此技能生效后将不再能使用。如果封禁到有批斗能力的 826，则整个 826 阵营当晚无法批斗。（可以连续封禁同一名玩家，生效为止）

注意

因为【封禁】优先于其他技能，该角色会在夜间率先睁眼。

【封禁】技能无法对被动获得信息的玩家（如零度君）生效，也无法对其他有封禁技能的玩家（如王金甲）生效。

# 丧葬一条龙

  • 常客： 每晚你将知道离你最近的 826 到你的距离。

# 乐山大佛

  • 佛光： 锁定技 你可以承受两次批斗。

# 晾衣杆

  • 不屈： 觉醒技 当任何 826 阵营的特殊技能（即除批斗以外的技能）对你生效前，免除之并使该 826 出局，然后你失去技能“反制”。
  • 反制： 锁定技 在技能【不屈】发动前，因任何原因出局，则下一个夜晚 826 阵营不能批斗。

# 听漏工

  • 聆听： 每天夜晚可以选择一名玩家，主持人会告知你该名玩家及其后一个未出局玩家中是否存在 826。（按圆桌序，序号最大的玩家的后一个是序号最小的玩家）

# 管道工

  • 施工： 每天夜晚可关闭一名角色所处的井盖进行施工，使其被关在井里，免受批斗、大字报、扣饭的攻击和推荐信的效果；如果管道工连续两天对同一名角色的井盖施工，其会因被困死在井里而出局。

# 管理人员

  • 清理垃圾： 限定技 夜间开始时，你可以指定一名玩家，使该玩家为 826 当晚的唯一批斗目标。（826 在批斗阶段会得知这一消息，且他们不可不批斗）

# 杨正贤

  • 受冤： 锁定技 你被查验时身份显示为 826，但被诈骗时不出局。
  • 脆弱： 锁定技 被邱白林守护或被周书记推荐都会使你直接出局。
  • 师恩： 锁定技 826 的批斗对你无效，但 826 阵营的特殊技能可以使你出局。
  • 双边： 锁定技 你的出局为826阵营获胜的必要条件。

# 香椿树

  • 魔法： 每天晚上可以指定两名玩家，交换其在夜间受到的效果(包括查验、守护、批斗、扣饭等)。可以指定自己为两名玩家之一。不能连续两晚重复指定其中任意一名玩家。

# 莉莉艾

  • 交换： 限定技 夜间你可以将你所在的阵营（精确到 132 子阵营）和任意一名玩家的阵营交换，使游戏的结束条件按照交换之后的判定。

# 极海听雷

  • 寻证： 锁定技 若场上没有【成电专家】，每当 826 使用技能时（含被动效果），你获得 1 个【罪证】。夜间开始时，你可以得知你当前的【罪证】数量（当晚 826 的【罪证】要下一晚才会更新），然后可以发动以下两个技能之一。
  • 明查： 当你第 次使用这一技能时，你可以消耗 个罪证，查验一个人的阵营。每晚最多使用一次，且不能与【揭露】同时使用。
  • 揭露： 当你第 次使用这一技能时，你可以消耗 个罪证，并选择一个正整数 ，得知目前场上存在的第 名 826 的位置（ 为目前场上的 826 数）。每晚最多使用一次，且不能与【明查】同时使用。

说明

场上没有【成电专家】意味着以下两种情况：

  1. 板子配置中有【成电专家】，但【成电专家】已经出局。
  2. 板子配置中没有【成电专家】。

# 涌雷电子

  • 咏雷： 限定技 白天发言阶段你可以翻牌，并且当场创作一首咏雷诗。若如此做，则你在下一个夜间无法被 826 批斗。

# 反雷电子

  • 反雷： 限定技 白天发言阶段你可以翻牌，并且当场创作一首反雷诗。若如此做，则 826 在下一个夜间只能批斗你；若你在下一个夜间前已经出局，则 826 无法批斗任何人。

注意

若涌雷电子翻牌，则当日反雷电子不能翻牌。反之亦然。

# 梦中人

  • 惊醒： 锁定技 你被 826 成功批斗后延迟一天出局，然后立即拥有【思维】技能。
  • 思维： 觉醒技 在【惊醒】触发的当晚，你可以询问主持人一个有关场上玩家身份或技能使用情况的是非问题。

# 厂长

  • 尊长： 锁定技 你是明牌 132，你开局拥有【厂长钢印】。

# 木叶飞舞

  • 审判： 限定技 夜间选择一名玩家，你与该玩家的身份将于白天开始时揭晓。若该玩家为 826，则该玩家出局，出局时不可发动任何技能；若该玩家为 132，则你和该玩家一起出局，出局时不可发动任何技能。

# 罗礼思

  • 假猴： 锁定技 你始终被视为一张【雷绍武】牌。
  • 扮猴： 你可以加入 826 群，与有批斗能力的 826 见面，并正常行使批斗功能。

注意

有【罗礼思】的板子，引入 132 和 826 各自的阵营技，以保证平衡性。

阵营技： 一局游戏只能发动一次，发动条件由投票决定的技能。

  • 132 阵营技发动条件： 在白天，不小于存活玩家的一半选择发动。
  • 826 阵营技发动条件： 在黑夜，不小于存活826的一半选择发动。

有【罗礼思】的板子中，可引入以下阵营技：

  • 惩罚： 826 阵营技 使一名 826（含罗礼思）显示出真实的身份。若为罗礼思，826 当晚增加一次批斗次数；若不是罗礼思，826 当晚减少一次批斗次数。
  • 揭露： 132 阵营技 使一名玩家显示出其真实的身份。

# 刘彻东（仅限双身份）

  • 大字报： 限定技 如果你被批斗或驱逐出局，可以写出大字报使另一名角色被驱逐。但是如果被扣饭出局，则不能使用该技能。
  • 说不准： 锁定技 每当白天开始时，若你的第一张身份未出局，你交换两张身份牌的位置。
