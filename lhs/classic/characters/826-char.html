# 826 阵营角色

# 雷绍武

  • 批斗： 每晚可以与其他有批斗能力的 826 阵营玩家一起选择一名玩家进行批斗，令其出局。
  • 摸零线： 限定技 白天可以摸零线使自己出局，然后直接进入黑夜。

说明

若无特殊说明，所有 826 阵营角色默认拥有技能【批斗】。“特殊技能”指除【批斗】以外的 826 阵营角色的技能。

# 石安迪

  • 批斗： 每晚可以与其他有批斗能力的 826 阵营玩家一起选择一名玩家进行批斗，令其出局。
  • 串联： 限定技 白天可以摸零线并将另一人串联入零线，使其和自己一起出局，然后直接进入黑夜。

# 雷副教授

  • 批斗： 每晚可以与其他有批斗能力的 826 阵营玩家一起选择一名玩家进行批斗，令其出局。
  • 大字报： 限定技 如果你被批斗或驱逐出局，可以写出大字报使另一名角色被驱逐。但是如果被扣饭、被诈骗或被质疑出局，则不能使用该技能。

说明

若场上同时存在老卞，则其他玩家只能知道某玩家出局后发动了【大字报】技能，不知道该玩家是老卞还是雷副教授。

注意

老卞或雷副教授出局后不能发动【大字报】技能的情形很多，按照板子规则自行判断即可。其他类似技能同理。

# 王为民

  • 批斗： 每晚可以与其他有批斗能力的 826 阵营玩家一起选择一名玩家进行批斗，令其出局。
  • 纠缠： 每个夜晚可以选择一名角色进行粒子纠缠，第二天夜晚前，若你出局，则被纠缠的角色也随之出局。但如果你因被鱼箱质疑或自我批斗而出局，则无法发动此技能。不能连续两晚上纠缠同一名角色。

# 天龙

  • 批斗： 每晚可以与其他有批斗能力的 826 阵营玩家一起选择一名玩家进行批斗，令其出局。
  • 自大/愚昧： 觉醒技 当你白天被投票指定为驱逐对象时，你亮明身份而不被放逐，此后你可以在夜间发言，但白天失去投票资格。

说明

若场上同时存在第一种人，则其他玩家只能知道某玩家“亮明身份免除放逐”，不知道该玩家是第一种人还是天龙。

# 鬼见怕

  • 批斗： 每晚可以与其他有批斗能力的 826 阵营玩家一起选择一名玩家进行批斗，令其出局。
  • 恐吓： 限定技 白天可以摸零线使自己出局，然后直接进入黑夜，且当晚所有 132 不能发动技能。

# 何文涛

  • 胆小： 锁定技。你不与 826 见面，直到其他 826 全部出局后才可以批斗。
  • 作法： 每晚可以通过作法获取一名玩家的具体身份。（如果拥有技能【批斗】，则先作法再批斗）

# 何松昂

  • 批斗： 每晚可以与其他有批斗能力的 826 阵营玩家一起选择一名玩家进行批斗，令其出局。
  • 精神作法： 限定技 你可以逆转成电专家和杜自誊的查验结果。

# 猴山 7

  • 挺雷： 锁定技 你知道所有 826 成员的位置及具体身份，但其他 826 不知道你的位置。初始状态下你无法参与批斗。
  • 伪装： 锁定技 你第一次被成电专家查验为 132。
  • 接力： 觉醒技 当场上其他 826 都出局时，你拥有【批斗】能力。

# 猴山 8

  • 批斗： 每晚可以与其他有批斗能力的 826 阵营玩家一起选择一名玩家进行批斗，令其出局。
  • 伪装： 锁定技 你第一次被成电专家查验为 132。

# 王金甲（原版）

  • 批斗： 每晚可以与其他有批斗能力的 826 阵营玩家一起选择一名玩家进行批斗，令其出局。
  • 封禁： 从第二天晚上起，每晚可以封禁一名角色，使其当晚无法发动任何技能。如果封禁到有批斗能力的 826，则整个 826 阵营当晚无法批斗。

注意

因为【封禁】优先于其他技能，该角色会在夜间率先睁眼。其间可以与 826 阵营讨论。

【封禁】技能无法对被动获得信息的玩家（如零度君）生效，也无法对其他有封禁技能的玩家（如吧主）生效。

# 王金甲（削弱）

  • 批斗： 每晚可以与其他有批斗能力的 826 阵营玩家一起选择一名玩家进行批斗，令其出局。
  • 封禁： 触发技 夜间你可以选择一名玩家，使其当晚无法发动任何技能。此技能生效后将不再能使用。如果封禁到有批斗能力的 826，则整个 826 阵营当晚无法批斗。（可以连续封禁同一名玩家，生效为止）

注意

因为【封禁】优先于其他技能，该角色会在夜间率先睁眼。其间可以与 826 阵营讨论。

【封禁】技能无法对被动获得信息的玩家（如零度君）生效，也无法对其他有封禁技能的玩家（如吧主）生效。

# 郭德胜

  • 批斗： 每晚可以与其他有批斗能力的 826 阵营玩家一起选择一名玩家进行批斗，令其出局。
  • 甜甜圈： 锁定技 若你未出局，826 阵营只能指定 826 分子或与 826 分子相邻的角色为批斗目标。（相邻指存活玩家按编号圆排列的相邻位置）
  • 视而不见： 觉醒技 你出局后，任何 132 的特殊技能对 826 无效。

说明

当技能【视而不见】触发后，若成电专家查验 826，结果为“查验无效”。

# bnllm

  • 鬼画符： 锁定技 你不与 826 见面，直到发动【】后才可以批斗。
  • ： 限定技 投票阶段私聊主持人发动，你投票的对象为驱逐的对象，然后本夜与 826 见面并获得技能【批斗】。

# 段闲香

  • 批斗： 每晚可以与其他有批斗能力的 826 阵营玩家一起选择一名玩家进行批斗，令其出局。
  • 瞎骂： 限定技 白天可以摸零线使自己出局，然后不经过发言直接开始一轮投票，每名玩家只能指定自己或与自己相邻的角色为目标，不能弃票，得票最高的角色出局，然后直接进入黑夜。若以此法使 826 阵营和其他阵营同时达到胜利条件，则 826 阵营单独胜利。

注意

可能出现平票情况。按板子规定采取“平票者全部出局”或“由段闲香在平票玩家中选择一人出局”中的一种处理方式。

# 三小

  • 批斗： 每晚可以与其他有批斗能力的 826 阵营玩家一起选择一名玩家进行批斗，令其出局。
  • 小号： 你投票时拥有额外的 2 票（8/9/10 人局）或 3 票（12 人局），你可以任意分配额外票数的使用，须在投票前告知主持人。

# 雷猴

  • 进盒： 锁定技 你与参与批斗的 826 队友见面，但不能单独批斗。
  • 出盒： 限定技 在白天发言阶段，你可以展示你的身份牌并指定一名玩家使其出局。

# 古往今来为宙乎

  • 批斗： 每晚可以与其他有批斗能力的 826 阵营玩家一起选择一名玩家进行批斗，令其出局。
  • 无中生有： 限定技 夜晚行动时，你可以指定一名 132 阵营玩家，转换其所在的子阵营。

# 邦钦孙

  • 批斗： 每晚可以与其他有批斗能力的 826 阵营玩家一起选择一名玩家进行批斗，令其出局。
  • 失重： 锁定技 若你在夜晚被一名 132 使用技能指定为唯一目标，你得知并记录此 132 的技能和身份；你在夜间无法出局。
  • 野爹榜： 限定技 白天可以摸零线使自己出局，然后令所有被【失重】记录的角色出局，然后直接进入黑夜。

# 数学诡异

  • 批斗： 每晚可以与其他有批斗能力的 826 阵营玩家一起选择一名玩家进行批斗，令其出局。
  • 悖论： 限定技 夜间你可以指定一名玩家的序号为 ；若已经有指定过 ，则在之后的夜间可以指定一名玩家的序号为 。此效果持续到白天开始。与一般的限定技（使用一次）不同，这是一个能使用两次的限定技。

说明

  1. 不与 826 见面的指向性技能不能以 号为目标；如果目标的序号当晚为 ，则该技能浪费，没有效果（特别地，成电专家等有查验技能的玩家查验 号为 132）。
  2. 826 可以批斗 号（或 号）目标，若如此做，周书记会被告知今晚被批斗的对象是 号（或 号）。
  3. 计算距离时， 号目标与所有人的距离视为 。

# 三江方士·黑化

  • 批斗： 每晚可以与其他有批斗能力的 826 阵营玩家一起选择一名玩家进行批斗，令其出局。
  • 扣帽： 限定技 夜间你可以猜测一名玩家的身份（四无人员除外），若猜测正确，则当晚可批斗玩家数 +1，否则 -1。

# 滕维建

  • 偏执： 锁定技 你不与其他 826 见面。
  • 滕语： 限定技 夜间开始时，你选定一人，则其当晚的所有技能由你代为使用。

# 王东辉

  • 偏执： 锁定技 你不与其他 826 见面。
  • 投稿： 每天夜间选择一名玩家进行「投稿」。若该玩家为 826，则你于下一夜加入 826 群，并将【投稿】技能替换为【批斗】技能；若该玩家不为 826，则你获得他的具体身份。

# QQ 雷绍武

  • 讲学： 任何时候你都可以在 826 群发言。
  • 装死： 限定技 白天开始时，你可以进入装死状态，然后获得技能【批斗】。

说明

装死状态显示【已出局】，不能在雷猴杀群里发言，不能投票。QQ 雷绍武存在的板子中允许所有人对显示已出局的人投票或使用技能。

# 真·雷绍武

  • 真雷： 锁定技 你的身份始终显示为 「826号 雷绍武」，只有真正的雷绍武才能拥有此身份。
  • 进盒： 锁定技 你在任何时候不能自己发言、投票或使用技能。
  • 傀儡： 夜晚开始时，可以由一名其他 826 选择控制你。然后直到下一晚开始时，其除批斗外的技能转移至你身上，他可以代替你使用技能、发言和投票。当你成为最后一个826时，其他出局的 826 可以代替你进行批斗。

# 宣传员

  • 支援： 锁定技 你不与 826 见面，成电专家查验你为 132，你出局与否不影响 132 胜利判定。
  • 劝降： 锁定技 开局时你最先知道身份，然后选择一个盲盒，将盲盒中的身份牌替换为【雷绍武】。选择此盲盒的人知道你的位置和被替换掉的身份。
  • 牺牲： 限定技 夜晚开始时，你可以让自己在天亮时出局，使这一晚其他 826 免疫出局效果。

# 826 先锋

  • 批斗： 每晚可以与其他有批斗能力的 826 阵营玩家一起选择一名玩家进行批斗，令其出局。
  • 冲锋： 限定技 你可以令本晚 826 批斗次数 +1，下一晚 826 批斗次数 -1；或者令本晚 826 批斗次数 -1，下一晚 826 批斗次数 +1。

# 最爱春雷 2021（仅限双身份）

  • 批斗： 每晚可以与其他有批斗能力的 826 阵营玩家一起选择一名玩家进行批斗，令其出局。
  • 伪装： 锁定技 即使你被置于第二张牌，你也可以发动批斗技能。
