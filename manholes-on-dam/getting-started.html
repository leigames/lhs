# 快速开始

下面介绍《井满田坝》的基本规则。

# 游戏进程

# 夜晚阶段

让所有玩家闭眼，然后进入黄昏。 在夜晚，玩家仍可自由发言，但是说书人应该禁止玩家聊到与当晚有关的任何事情，也应该建议玩家不在首个夜晚针对自己的角色进行发言。

如果这是首个夜晚，并且为 7 人及以上游戏，进行下面两个步骤：

  1. 同时唤醒所有的团员，告诉他们谁是团长。
  2. 唤醒团长，告诉他谁是团员。然后给他展示三个不在场的 132 角色。注意：826 没有夜晚的群聊。

根据夜晚顺序表的顺序，依次唤醒每一个能醒来使用角色能力的玩家。

# 白天阶段

1. 白天的交流

在白天，玩家可以以任何形式交流信息。如果私聊，将自己想要私聊的人与说书人拉入一个单独的群中。说书人在私聊的时候会告知所有玩家：哪些玩家进行了私聊（相隔一段时间再进行也需告知，由说书人决定）。

2. 提名与投票

由说书人宣布白天进入提名环节。需要确保所有玩家都回到了原本的座位上。

提名有以下几个限制： 同一时间只能有一名玩家被提名。

如果一项提名已经被发起但处决投票尚未结束，则不能再提出另一项提名。

只有存活的玩家可以发起提名。虽然进盒的玩家也可以被提名，但这在绝大多数情况下都不是一个明智的举动。

每名玩家每天只能发起一次提名，并且每名玩家每天只能被提名一次。

在投票之前，被提名的玩家有为自己辩护的机会。随后会进行投票统计。

在统计投票时： 每名存活的玩家每天可以投票给任意数量的玩家（投给同一位玩家只能一票）。

每名进盒的玩家在他进盒后，只剩最后一次投票机会。

在计票结束之后，会宣布本次投票结果。

投票在以下条件均满足时成功：

  • 被提名的玩家获得的票数在今天所有被提名的玩家中最多。（不得是与其他人同为最多）
  • 他的票数等于或超过了存活玩家人数的一半。

此时玩家会进入“即将被处决”状态。

投票后，说书人公布该玩家得票，但不进入黑夜。

如果说书人觉得当天的提名和投票已经足够多了，或无人再发起提名，可以为当天的提名给出最后一次机会，并在读秒结束后宣布提名阶段结束。

3. 处决与进盒

在处决阶段，当前“即将被处决”的玩家会被处决。说书人会宣布这名玩家进盒（或宣布这名玩家被处决但没有进盒，如果他因为任何效果导致不死的话）。

每个白天只能有一名玩家被处决。在处决结束后，白天阶段随即结束。

并非每个白天都需要处决一名玩家。如果玩家们不打算提名任何人，或是被提名的玩家没有得到足够的票数，亦或者是最高得票出现了平局，这个白天都会以无人被处决而结束。

# 游戏结束

一旦有阵营达成胜利条件时，游戏结束。

132 阵营的胜利方式：所有团长均进盒。

826 阵营的胜利方式：场上只剩两名玩家存活。（猴山游客的数量不计算在内）

如果两个阵营同时达成胜利条件，132 阵营获胜。

# 身份配置

按人数决定各阵营身份数量。（初始状态下，四有人员、四无人员同属 132 阵营，团员和团长同属 826 阵营）

5、6 人时，没有团员信息和团长信息。

# 状态

# 阵营与角色

在任意时间点，一名玩家一定会处于 132 或 826 两种状态之一。玩家的角色决定了玩家初始所属的阵营是 132 还是 826，但玩家的阵营可能会在游戏过程中发生变化：此时，一名 132 阵营的玩家可能拥有一个 826 角色，一名 826 阵营的玩家也可能拥有一个 132 角色。

角色与阵营这两个状态之间是相对独立的。如果一名玩家改变了阵营，他的角色会保持不变，反之亦然。例如，如果第一种人变为 826，他仍然还是第一种人。又例如，假证贩子将 132 阵营的 nideqiancou 变成了郭德胜，这位郭德胜仍然属于 132 阵营。

在游戏过程中，如果玩家的阵营或角色发生变化，该玩家会在第一时间秘密得知这一变化内容。

得知自己的角色或阵营发生变化并不算是获得信息。因此也不会受到看不清楚、精神错乱或者类似何松昂这样的角色所影响。这是为了确保玩家都能正确地得知自己的阵营和角色。当然，也会有一些例外。有时玩家会以为自己是与自己真实角色不同的角色，或是与自己真实阵营不同的阵营。例如，迎风乐回和谦卑的九头蛇的设计就是为了让玩家不知道自己的真实角色是什么。

# 精神错乱与看不清楚

非信息技能发动必定失败（不会得知），信息类技能大概率失败（说书人决定，看平衡状况）。

说明

在任意时间点，一名玩家一定会处于看得清楚或看不清楚两种状态之一，也一定会处于精神错乱或精神正常两种状态之一。如果一名玩家在他看不清楚或精神错乱期间尝试使用自己的“每局游戏限一次”的能力，他无法在之后再次使用这项能力。使用机会会被浪费。看不清楚或精神错乱的玩家能有机会恢复能力。如果一名看不清楚的玩家恢复看得清楚，或一名精神错乱的玩家恢复精神正常，那么他就能恢复自己的能力。即便如此，如果他已经使用了自己的 “每局游戏限一次” 能力，很不幸，他仍无法再次使用这项能力。不告诉玩家他看不清楚或是精神错乱了！取而代之的是，让他们以看得清楚和精神正常的状态来执行行动。看不清楚或精神错乱的玩家，其角色能力可能会收到错误的信息。看不清楚与精神错乱状态不会互相抵消。对看不清楚或精神错乱的玩家所使用的能力仍然会正常生效。即使玩家看不清楚或精神错乱，说书人也要保持给他们提供正确的规则。如果一项能力是持续型的能力或者正在游戏中产生效果，玩家会在看不清楚或精神错乱时失去这些能力，但会在恢复看得清楚和精神正常时继续执行这些能力的效果。

# 疯狂表现

正在尝试说服他人相信某件事情。

说明

判断一名玩家是否疯狂表现的决定权在说书人手中。规则中没有硬性规定玩家哪些可以说，哪些不能说，哪些必须得说，哪些一定不能说。每局游戏都可能会出现不同的情况。关键的是，说书人认为这个玩家是否：表现得非常努力；或真诚地证明某件事情；或让自己给出的信息引发了讨论，人们认为他的信息值得被接受；或其他玩家单纯只是相信这名玩家；或等等其他情况。精准地去判断玩家是否疯狂表现，尤其是玩家的行为和语言是否符合疯狂表现这一行为并不是一件能够被精确的状态和硬性的规则所能规定的。

同时，一般带有“疯狂表现”的角色技能是否生效的决定权也在说书人手中。假如一名玩家获得了 nideqiancou 这个角色，并直接说明自己是这个角色，那么说书人可以判定这名玩家不被处决，因为这将直接证明他的身份。

任何一名玩家都可以决定自己是否“疯狂表现”。每一名玩家都是可以在任何时候说任何想说的话的，他们从来不会被迫说出任何不想说的话。每一个运用了疯狂表现机制的角色都以独特的方式来实现了这一点，而疯狂表现对以特定方式说（或是不说）特定事情有着特定的惩罚或好处，但玩家永远不会被强迫去说任何事情。
