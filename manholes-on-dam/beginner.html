# 初识民科

《初识民科》即官方的《暗流涌动》剧本，属于新手剧本。

可查看《钟楼百科》的有关内容：剧本-暗流涌动 (opens new window)

# 四有人员

# 成电专家（占卜师）

每个夜晚，你要选择两名玩家：你会得知他们之中是否有团长。

会有一名 132 玩家始终被你的能力当作团长。

# 零度君（送葬者）

每个夜晚（第一夜除外），你会得知今天白天死于处决的玩家的角色。

# 邱白林（僧侣）

每个夜晚（第一夜除外），你要选择除你以外的一名玩家：当晚团长的负面能力对他无效。

# 梦中人（守鸦人）

如果你在夜晚进盒，你会被唤醒，然后你要选择一名玩家：你会得知他的角色。

# Mono6（贞洁者）

当你首次被提名时，如果提名你的玩家是四有人员，他立刻被处决。

# 老卞（猎手）

每局游戏限一次，你可以在白天时公开选择一名玩家：如果他是团长，他进盒。

# 刘淑秋（士兵）

团长的负面能力对你无效。

# 厂长（镇长）

如果只有三名玩家存活且白天没有人被处决，你的阵营获胜。

如果你在夜晚即将进盒，可能会有一名其他玩家代替你进盒。

# 虎爪吃布丁（图书管理员）

在你的首个夜晚，你会得知两名玩家和一个四无人员角色：这两名玩家之一是该角色（或者你会得知没有四无人员在场）。

# 成电专家的小号（洗衣妇）

在你的首个夜晚，你会得知两名玩家和一个四有人员角色：这两名玩家之一是该角色。

# 极海听雷（调查员）

在你的首个夜晚，你会得知两名玩家和一个团员角色：这两名玩家之一是该角色（或者你会得知没有团员在场）。

# 管理人员（厨师）

在你的首个夜晚，你会得知场上邻座的 826 玩家有多少对。

# 听漏工（共情者）

每个夜晚，你会得知与你邻近的两名存活的玩家中 826 玩家的数量。

# 四无人员

# 科学奴（圣徒）

如果你死于处决，你的阵营落败。

# 杨正贤（陌客）

你可能会被当作 826 阵营、团员角色或团长角色，即使你已进盒。

# 迎风乐回（酒鬼）

你不知道你是迎风乐回。你以为你是一个四有人员角色，但其实你不是。（因为迎风乐回并非他认为的角色，所以他得到的信息说不准，非信息类技能无效；迎风乐回始终会被视为处于【看不清楚】状态）

# 川大新生（管家）

每个夜晚，你要选择除你以外的一名玩家：明天白天，只有他投票时你才能投票。

# 团员

# 猴山 7（间谍）

每个夜晚，你可以查看说书人记录。

你可能会被当作 132 阵营、四有人员角色或四无人员角色，即使你已进盒。

# 燦燦火龙飞（红唇女郎）

如果大于等于五名玩家存活时（猴山游客不计算在内）团长进盒，你变成那个团长。

# 李子丰（男爵）

会有额外的四无人员在场。 游戏初始配置改变：[+2 四无人员]

# 滕维建（投毒者）

每个夜晚，你要选择一名玩家：他在当晚和明天白天精神错乱。

# 团长

# 雷绍武（小恶魔）

每个夜晚（第一夜除外），你要选择一名玩家：他进盒。如果你以这种方式使自己进盒，一名团员会变成雷绍武。

# 加强版角色

# 月无影（无神论者）

（四有人员）

说书人可以打破游戏规则。如果说书人被处决，132 阵营获胜，即使你已进盒。 游戏初始配置改变：[无 826 角色在场]

# 谦卑的九头蛇（疯子）

（四无人员）

你以为自己是一个团长，但其实你不是。

团长知道你的身份以及你每晚选择的玩家。

# 鬼见怕（军团）

（团长）

每个夜晚（第一夜除外），可能有一名玩家进盒。如果一项提名只有 826 玩家投票，投票无效。你也会被当作是团员。 游戏初始配置改变：[半数以上玩家为鬼见怕]

# 行动顺序

5、6 人时，没有团员信息和团长信息。

  • 第一夜： 团员获得信息-团长获得信息-滕维建-成电专家的小号-虎爪吃布丁-极海听雷-管理人员-听漏工-成电专家-川大新生-猴山 7
  • 其他夜晚： 滕维建-邱白林-燦燦火龙飞-雷绍武-鬼见怕-梦中人-听漏工-成电专家-川大新生-零度君-猴山 7
